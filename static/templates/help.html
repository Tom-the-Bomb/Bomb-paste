API Usage
POST /upload

Simply uploads a paste, accepts JSON payload data with a content field specifying the paste content and returns JSON response with an id field specifying the paste ID. The save button also just calls this endpoint internally.

Limits:  content length must be between {{ min_content_length }} and {{ max_content_length }} characters & Max {{ max_upload_rate }} reqs per {{ max_upload_per }}s
Example Response:  {"id": "gEDjNieoN3LsFCWTJIEc"}
Example Request: 

POST /upload
Content-Type: application/json
body: {
    "content": "print('hello world')"
}
